READ ME: HIT AUTOMATON

Version:

---------------------------------------------------- Requirements ----------------------------------------------------

-Java 7

Note: Some features not compatible with MacOS

------------------------------------------------------ How To Run -----------------------------------------------------

From Eclipse: Select main project folder and run as Java application

From .exe: Double click on the application, it should start up fairly quickly

-------------------------------------------- Creating a New Document --------------------------------------------

Go to File->New.

Note: Upon startup the application will have a clean document.

----------------------------------------------- Opening a Document ------------------------------------------------

Go to File->Open

Navigate to the directory the file is saved in

Select the file and push ‘open’

Note: The application only supports .fa files

------------------------------------------------ Saving a Document ------------------------------------------------

Go to File->Save As.. if the file has never been saved before

OR

Go to File->Save if the file has been saved previously

Pick a name for the file to be saved as

Press ‘Save’

Note: The file will be saved as both a .fa and a .xml document. The .xml document will be saved as nameXML.xml.

----------------------------------------------- Exiting the Application -----------------------------------------------

Make sure to save!

Press the ‘X’ in the corner

OR

File->Quit

---------------------------------------------------------- Help ----------------------------------------------------------

Go to Help->Help for more information on the application’s features.
